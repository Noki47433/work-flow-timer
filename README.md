# Work Flow Timer

A minimal, distraction-free work timer with a built-in todo notepad.

## Features
- Start / stop work timer
- Tracks total time worked today
- Paste todos and check them off
- See how fast each task was completed
- Clears the list when all tasks are done
- Shows a motivational quote at the end of the day

## Usage
Just open `index.html` in your browser.

## Why
Built to stay boring, focused, and consistent.
